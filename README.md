Coffee Shop Website

A simple and visually appealing Coffee Shop website built using **HTML** and **CSS**. This project focuses on creating an attractive user interface for a coffee shop landing page.

Live Demo
Visit the website here:
https://kanikatomar18.github.io/Coffee-Shop/

Features
* Attractive coffee shop-themed design
* Clean and modern user interface
* Multiple sections for showcasing content
* Easy navigation
* Pure HTML and CSS implementation
* Beginner-friendly project structure

Technologies Used
* HTML5
* CSS3

Project Structure

```
Coffee-Shop/
│
├── index.html
├── style.css
├── about.html
├── style-about.css
├── menu.html
├── style-menu.css
├── contact.html
├── style-contact.css
├── images/
└── README.md
```

Purpose of the Project
This project was created to practice:
* HTML page structure
* CSS styling and layouts
* UI design principles
* Front-end web development fundamentals

How to Run
1. Clone the repository:
```bash
git clone https://github.com/kanikatomar18/Coffee-Shop.git
```
2. Open the project folder.
3. Run `index.html` in any web browser.

Preview
The website presents a stylish coffee shop landing page with an attractive layout and coffee-themed design elements.
